# API Contracts

Этот репозиторий содержит OpenAPI-спецификацию для системы автоматизации заказов ресторана быстрого питания с роботизированным конвейером.

## Просмотр спецификации

Вы можете посмотреть спецификацию в Swagger UI, скопировав содержимое файла `openapi_project-otus.yaml` в [Swagger Editor](https://editor.swagger.io/).

## Структура

- `openapi_project-otus.yaml` — основная спецификация API.
